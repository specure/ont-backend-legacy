All providers have to be defined here. The entries inside this table ARE NOT automatically assigned to test results. To achieve a correct assignment both tables: as2provider and mccmnc2provider need to be set up correctly as well. Additionally it is possible to define map filters for all NetTest applications by setting the map_filter flag. Only providers with a valid value in column mcc_mnc will be displayed in the list for mobile operators.
